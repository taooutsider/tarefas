# Decisions

## 2026-06-06 - Scope broadened to cross-subnet mining viability
Tiger Miner is no longer scoped only to SN94 BitSota. The active objective is to find, evaluate, rank, and execute viable Bittensor subnet mining opportunities with focus on net profitability, low wasted spend, wallet safety, and low-to-moderate infrastructure.

Operational gate before recommending live mining:
- active emissions now
- active miner/validator activity now
- registration open or access path realistic
- repository/docs sufficient to run and compete
- required APIs/tools obtainable
- infrastructure cost bounded
- wallet/hotkey/register path safe
- expected reward path defensible against recurring cost and capital at risk
